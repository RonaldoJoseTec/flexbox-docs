Flexbox

Documentação

É o recurso do CSS o qual nos permite organizar elementos em linha ou coluna; Mas antes de tudo precisamos compreender dois conceitos imprescindíveis que são:

Container: é o elemento principal da estrutura local onde serão organizados os items

Items: são os elementos que deverão ser alocados no container - filhos do container -

Display: flex

A primeira propriedade que precisa ser declarada para que o Flexbox funcione!

Desta forma conseguimos organizar o layout dos items usando as diversas propriedades, com isso demonstraremos as suas definições a seguir.

"Flex-start"

os items se alinham à esquerda do container

"Flex-end"

os items se alinham à direita do container

"Space-Around"

os items são exibidos com espaço antes, entre e depois das linhas

"Center"
